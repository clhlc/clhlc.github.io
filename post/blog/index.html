Github+hugo 搭建个人博客

Apr 18, 2024
1 minute read

# Hugo安装

# 基于MacOS安装 

brew install hugo

# 新建一个博客目录 

hugo new site myblog

# 配置博客主题

主题可以在 Hugo Themes 网站上找到, 本人使用hugo-theme-stack主题

cd myblog
git init
git submodule add https://github.com/CaiJimmy/hugo-theme-stack.git themes/hugo-theme-stack

# 创建文章

Hugo 中的文章都以 Markdown 格式写作。在 content/post 目录下新建一个 Markdown 文件

hugo new post/post01.md

# 本地预览文章 

hugo server -D

  2. 主题在 themes/ 目录下
  3. hugo.toml 文件中指定使用的主体名字，名字和 themes 目录下的主体名字一致

# 我的Blod工作流程

  1. 再新建一个私有的github repo
  2. 提交myblog到私有repo
  3. 配置github action，自动生成public目录，并把public目录提交到xxxx.github.io对应的github public repo
